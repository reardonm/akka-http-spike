Akka-HTTP Microservice Spike
======

Akka with Spring in Java 8, using Maven.

| Layer      | Impl      |
| ---------- |:----------|
| HTTP       | Akka-HTTP |
| REST/API   | Akka-HTTP |
| JSON       | Jackson   |
| DI         | Spring Framework |
| FSM        | Akka |
| Config     | Typesafe Config |
| Logging    | TBD |  
| HTTPClient | TBD |
